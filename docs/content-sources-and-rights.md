# 内容来源与版权边界

## 来源优先级

1. 教育部、教育考试机构和政府网站的正式文件。
2. 正式发布机构、官方出版社或权威教育研究机构。
3. 无官方在线全文时使用可靠数字图书馆、学术数据库、档案或正式影印本，并明确标注来源性质。

转载网页不能替代原始出处；搜索结果摘要不能作为正文证据。在线文本必须先确认篇目和版本，同文异版只能旁证作者、篇名等稳定事实。

## 仓库内容

公开仓库保存原创代码、方法文档、事实性元数据、来源链接、哈希和为核查所需的有限引文。不保存 Downloads 原始 PDF、本地历史汇编扫描件、整本 OCR 转录、Cloudflare 数据库导出、评论数据或 AI 对话内容。

源码与原创文档按根目录 `LICENSE` 授权。第三方文件、官方标准、出版物、扫描件和引文的权利仍归相应权利人；仓库中的元数据不构成对这些材料的再授权。

## 移除与修正

若权利人或发布机构认为某项内容不应公开，请通过 GitHub issue 提供公开资料的修正依据；涉及隐私、未公开内容或安全信息时使用 GitHub Security Advisory 私密渠道。确认后应优先移除公开副本而保留必要的来源、哈希和审计记录。
