# R6 外语受控文档同版在线核对源映射

审查时间：2026-07-16
范围：4 份俄语课标、1 份日语课标
结果：`publication_gate_changed=false`，`can_unlock=false`

本轮只建立证据映射，不修改 OCR、corpus、concept、前端、引用资格或发布门禁。判断严格区分：

- `exact_artifact`：在线文件与本地文件 SHA-256、PDF 页数完全相同。只能证明来源与文件身份；同一文件的搜索抽取不算独立文本。
- `same_edition_different_artifact`：同版但文件哈希或分页不同。可在完成页码映射后辅助核文，来源独立性未证明时仍须 fail-closed。
- `same_edition_translation`：明确依据同一原版制作的翻译或重排文本。可核结构、词条或释义，但不能替代原版图像逐行裁决。
- `different_edition`：版次、修订年或课程体系不同。只能提示版本差异，不能裁决可能变化的词表、能力要求或评价标准。
- `unconfirmed`：同版或独立性证据不足，不得用于修订、引用或解封。

该文件从原源 102 个 PDF 物理页重排为 78 页，不能按物理页一对一映射，但可用词条锚点定位：

- 本地 p61 ↔ 在线物理页 48-49：`こおり [氷] 冰`、`こくばん [黒板] 黑板`、`ここ 这里，这儿`、`ごご [午後] 下午`、`こんな 这样的`
